# MainGameFreeHMasturbationMenu

## Overview
- Adds a masturbation entry to the normal FreeH action menu.
- Makes masturbation poses reachable from the regular action flow without losing the path back to normal poses.

## What It Does
- Adds a custom action button to the H action UI
- Switches into masturbation pose selections
- Optionally cycles masturbation poses on repeated use
- Can keep the action menu visible during masturbation
- Includes transition recovery helpers

## Target Process
- `KoikatsuSunshine`

## Main Files
- `MainGameFreeHMasturbationMenu.dll`
- `MainGameFreeHMasturbationMenuSettings.json`
- `MainGameFreeHMasturbationMenu.log`

## Main Settings
- `Enabled`
- `FreeHOnly`
- `ButtonText`
- `ButtonOffsetX` / `ButtonOffsetY`
- `CycleMasturbationPoses`
- `KeepActionMenuVisibleInMasturbation`
- `AutoRecoverTransitionFromMasturbation`
- `VerboseLog`

## Notes
- This plugin targets FreeH menu flow rather than full pose management.
