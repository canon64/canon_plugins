# MainGameFreeHMasturbationMenu

## 概要
- FreeH の通常アクションメニューにオナニー項目を追加するプラグインです。
- 通常導線から直接オナニー体位へ入り、必要なら通常体位へ戻る流れも維持します。

## できること
- HアクションUIへの追加ボタン生成
- オナニー体位選択への切り替え
- 連続使用時の体位巡回
- オナニー中のアクションメニュー表示維持
- 遷移復帰補助

## 対象プロセス
- `KoikatsuSunshine`

## 主なファイル
- `MainGameFreeHMasturbationMenu.dll`
- `MainGameFreeHMasturbationMenuSettings.json`
- `MainGameFreeHMasturbationMenu.log`

## 主な設定
- `Enabled`
- `FreeHOnly`
- `ButtonText`
- `ButtonOffsetX` / `ButtonOffsetY`
- `CycleMasturbationPoses`
- `KeepActionMenuVisibleInMasturbation`
- `AutoRecoverTransitionFromMasturbation`
- `VerboseLog`

## 注意点
- 体位管理全体ではなく、FreeH メニュー導線の拡張が主目的です。
