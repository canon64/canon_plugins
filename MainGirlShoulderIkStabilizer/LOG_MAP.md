# MainGirlShoulderIkStabilizer ログマップ

## 先に結論（最短）
- 通常ログを見る場所:
  - `F:/kks/BepInEx/plugins/canon_plugins/MainGirlShoulderIkStabilizer/log/MainGirlShoulderIkStabilizer.log`
- 肩角度の内部計算ログを見るには:
  - `Logging.ShoulderDiagnosticLog = true`
- 診断ログを一発で全部止めるには:
  - `Logging.ShoulderDiagnosticLog = false`

## ログの出口（どこに出るか）
- 基本は `MainGameLogRelay` 経由で出力される。
- Relay が使えないときだけ BepInEx 標準ロガーへフォールバックする。
- このプラグインのログキーは `main/MainGirlShoulderIkStabilizer`。

## スイッチ一覧（調整で触る項目）
- `Logging.EnableLogs`
  - Relay経由ログの出力ON/OFF。
- `Logging.ShoulderDiagnosticLog`
  - 肩角度の内部計算ログを出すかどうか。
  - `false` なら診断ログは完全停止。
- `Logging.ShoulderDiagnosticLogInterval`
  - 診断ログの出力間隔（秒）。
  - 範囲: `0.05` ～ `2.0`。
- `General.VerboseLog`
  - 設定更新などの補助ログを増やす。

## 通常ログ文言の意味
- `起動完了`
  - プラグイン起動完了。
- `設定ファイル=...ShoulderIkStabilizerSettings.json`
  - 読み込んだ設定ファイルの場所。
- `有効状態: プラグイン=True 肩補正=True`
  - メインON/OFF状態。
- `対象女性=...`
  - 現在補正対象の女性キャラ名。
- `animBody=...`
  - 使用中のアニメーションBody名。
- `肩補正コンポーネントを接続: ...`
  - 肩補正コンポーネント接続完了。
- `肩補正コンポーネントを解除: ...`
  - 肩補正コンポーネント解除（理由付き）。
- `参照解決に失敗: ...`
  - 参照解決失敗（対象やIKが取れていない）。
- `設定ファイルを再読込`
  - JSON再読込が入った。

## [肩診断] の読み方（内部計算）
- `肩=左肩/右肩`
  - 左肩か右肩か。
- `腕下げ判定=はい/いいえ`
  - 腕が下がり判定か。
- `反転適用=はい/いいえ`
  - 下げ腕時の反転が適用されたか。
- `上限制限=はい/いいえ`
  - 1フレーム角度上限で制限されたか。
- `腕の上下差Y`
  - 腕先の上下差（キャラ基準）。
- `腕上げ進行度`
  - 腕上げ判定の進み具合（0～1）。
- `重み(設定→実効)` / `オフセット(設定→実効)`
  - 設定値から実際に使った値への変化。
- `腕の伸び率`
  - 腕の伸び率。
- `補正強度(raw/0-1/上限制限後/最終)`
  - 補正強度が最終値になるまでの段階値。
- `ソルバー重み(エフェクタ/IK全体)`
  - エフェクタ重み / IK全体重み。
- `角度(目標差/反転前/反転後/制限前/最終/上限)`
  - 角度決定の流れ:
  - `目標差`: 目標方向とのズレ角
  - `反転前`: 反転前の適用候補角
  - `反転後`: 反転後角
  - `制限前`: 上限制限前角
  - `最終`: 最終適用角
  - `上限`: 1フレーム上限角

## 調整の見方（実務）
1. `ShoulderDiagnosticLog=true` で開始する。
2. `角度(...最終/上限)` で `最終` が常に `上限` 付近なら強すぎの可能性が高い。
3. `腕下げ判定=はい` で不自然なら `反転適用` と `重み/オフセット` を確認する。
4. `最終` がほぼ `0` なら `補正強度(.../最終)` と `ソルバー重み` を先に確認する。
5. 方針が決まったら `ShoulderDiagnosticLog=false` で通常運用に戻す。
